# Student Grade Calculator (C++)

This application calculates students' final grades based on homework and exam results.

## Final Grade Formula

Final grade is calculated using:

Final = 0.4 * Homework_Average + 0.6 * Exam

OR

Final = 0.4 * Homework_Median + 0.6 * Exam

The user can choose which calculation method to use (Average or Median).

## Features (planned)

- Rule of Three implementation
- std::vector container usage
- Manual data input
- Random data generation
- File input (Students.txt)
- Sorting students by name or surname
- Formatted output
